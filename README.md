# 📦 Supply Chain Tracker (Soroban Smart Contract)

## 📌 Project Description
Supply Chain Tracker is a decentralized application built on the Stellar network using Soroban smart contracts. It allows tracking of products throughout their lifecycle — from origin to final delivery — ensuring transparency, trust, and tamper-proof records.

---

## 🚀 What it does
This smart contract enables users to:
- Register products on-chain
- Track ownership changes
- Maintain a transparent history of product transfers
- Verify product origin and authenticity

---

## ✨ Features

- 🔗 **Decentralized Tracking**
  All product data is stored on-chain, ensuring transparency and immutability.

- 📍 **Product Registration**
  Add new products with unique ID, name, and origin.

- 🔄 **Ownership Transfer**
  Transfer products between stakeholders in the supply chain.

- 📜 **History Tracking**
  Maintains a record of previous owners for full traceability.

- 🔍 **Product Lookup**
  Fetch complete product details anytime.
